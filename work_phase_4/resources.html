Supporting Resources

To ensure that all users can have the best experience, we provide the following supporting resources. These tools can help you better access and use our website.

Google Chrome

Google Chrome is a fast, secure, and user-friendly web browser suitable for all devices. It offers a wide range of accessibility features, including screen reader support, keyboard navigation, and customizable display options.

NVDA Screen Reader

NVDA (NonVisual Desktop Access) is a free and open-source screen reader that helps visually impaired users operate computers. It uses text-to-speech technology to read the text on the screen, enabling visually impaired users to access websites and applications.

Windows Keyboard Navigation

The Windows operating system comes with a variety of keyboard navigation features, such as Sticky Keys, Filter Keys, and Toggle Keys, which help users operate the computer more easily using the keyboard without relying on the mouse.

Dyslexie Font

Dyslexie is a font specifically designed for people with dyslexia. Through unique letter shapes and spacing, it makes text easier to read. This font can help people with dyslexia reduce reading errors and improve reading speed and comprehension.

Color Contrast Checker

The color contrast checker is a tool that helps designers and developers ensure sufficient contrast between the text and background colors of a website, making the content more readable for users with visual impairments.
